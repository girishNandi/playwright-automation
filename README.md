# playwright-automation
Playwright test automation project for UI testing of web application.

# This repository contains automated UI tests built using Playwright.

# Pre requisites

Install Node.js
npm

# Installation
Clone the repo:  https://github.com/girishNandi/playwright-automation/

# Install dependencies
- npm install
- npm init playwright@latest

# Running the tests
- Headless : npx playwright test
- Headed : npx playwright test --headed

# View Report 
npx playwright show-report

# Tools Used
- Playwright
- Node.js
- Git and GitHub
